# 第三方库文件夹

此文件夹用于存放不需要打包到主应用中的第三方JavaScript库文件。这些文件将被单独构建和加载，以优化应用的加载性能。

## 使用说明

1. 将需要单独构建的第三方库文件放置在此文件夹中
2. 每个库文件应该是独立的模块
3. 这些文件将不会被打包到主应用的bundle中
4. 可以通过动态导入(dynamic import)或CDN的方式加载这些文件

## 目录结构

```
libs/
  ├── README.md
  └── [library-name]/      # 每个第三方库单独一个文件夹
      ├── index.js
      └── ...
```
